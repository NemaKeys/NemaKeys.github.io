How to save an image stack (VCE video) using ImageJ (Fiji)

  1. Download Fiji.
  2. https://imagej.net/software/fiji/downloads
  3. Unzip the downloaded Fiji zip file.
  4. Double-click the executable to run Fiji. (fiji-win64.exe)
  5. Select the images you want to stack.
  6. Drag the images onto the Fiji Too bar.
  7. Process images to a stack
  8. Save stacked images as an AVI file.
  9. Use HandBreak to convert the AVI file to an MP3 so you can display the video on FaceBook, Twitter or any other HTML page.https://handbrake.fr/downloads.php
  10. Open Handbreak and drag and drop the AVI file.
  11. Press the Start Encode button.
  12. The file should now be saved as an MV4 and will play on an HTML page when clicked on
